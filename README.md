# Smart Task Planner

## Description

Smart Task Planner is a web-based application designed to help users organize and manage their daily tasks in an efficient and structured manner.

## Problem

Many individuals face difficulties in managing their time and prioritizing tasks effectively, which often leads to decreased productivity and missed deadlines.

## Solution

The application provides a simple interface that allows users to create, organize, and manage tasks based on their priority levels, enabling better time management and task tracking.

## Features

* Create new tasks
* Edit existing tasks
* Delete tasks
* Assign priority levels (High, Medium, Low)
* View tasks in an organized list
